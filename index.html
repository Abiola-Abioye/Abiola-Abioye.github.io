Project Journey

Retail Inventory Automation & Markdown Optimization

Power BI Python Data Modeling DAX Framework

The Business Problem: Frontline retail teams routinely execute manual shelf-rotation checklists to catch expiring inventory. This process is time-intensive, prone to human oversight, and often collides with peak front-end checkout traffic, leading to lost productivity and increased waste (shrinkage).

The Solution: Deployed a priority-driven, exception-based inventory tracking model. Engineered a custom 3-month retail transactional dataset using Python (pandas, numpy) and architected a flawless, normalized Star Schema database inside Power BI by abstracting a unique batch lookup dimension to break complex routing loops.

Relational Star Schema
Automated Operations UI

Strategic Impact: Completely replaces blind manual scanning with a predictive, targeted markdown picklist. By implementing automated time-intelligence thresholds, the dashboard surfaces high-risk stock and applies a tier-decay markdown strategy (25%, 50%, 75%) to maximize product salvage recovery while freeing up labor hours for high-traffic customer waves.
